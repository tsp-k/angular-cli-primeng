# Angular CLI + PrimeNG

Packages:
```
Angular 4.0.0
Angular CLI 1.2.7
PrimeNG 4.1.2
Font Awesome 4.7.0
```

Install the dependencies:
```
npm install
```

For sanity check run your new app at http:\\\\localhost:4200
```
ng serve
```

Production build:
```
ng build --prod
```

